# Todo List App

Многофункциональное веб-приложение для управления задачами с возможностью установки приоритетов, дедлайнов, фильтрации и переключения темы.

## Стек технологий

 **Frontend:** HTML5, CSS3, JavaScript (ES6+)
 **База данных:** localStorage
 **Деплой:** Vercel

## ✨ Функционал

 Добавление задач с приоритетом (высокий/средний/низкий)
 Редактирование по двойному клику
 Удаление задач
 Массовое удаление всех задач
 Фильтрация: Все / Активные / Выполненные
 Подсчёт активных задач
 Отображение времени создания
 Установка дедлайна с подсветкой просрочки
 Тёмная тема с переключением
 Автоматическое сохранение в localStorage
 Адаптивный дизайн
